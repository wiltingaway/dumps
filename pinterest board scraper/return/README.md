# return
this returns a random url from a board
# setup
firstly, you need get cookies from your pinterest profile  
download a [cookie editor](https://chromewebstore.google.com/detail/cookie-editor/hlkenndednhfkekhgcdicdfddnkalmdm) (any is fine, if you're on a chrome based browser use the url) and go on pinterest.com, allow cookie-editor & dump your cookies in the format of "Netscape"  
after that, make a cookies.txt inside of the same directory where you have the file and that's it!
# usage
`python main.py --name [pinterest username] --board [board name]`
optionally, if you want to return all of them  
simply at the end add `--all`
