# vsc timer
always wanted to track time of when you worked but discord presence was innacurate?  
here's a cool script that'll do it  
you can use anything really
# usage
you can use the default settings if you're using vsc and you think it's fine but i would recommend changing it depending really on how you like it (you can even replace the asii if you'd like text etc)  
`program=""` (vsc is "Code.exe")
`interval=` (on how often to update)
`idle_threshold=` (when you stop moving your mouse, after how long it should start tracking your offline time)
